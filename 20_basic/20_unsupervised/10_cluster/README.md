# 聚类分析（Cluster Analysis）

聚类是无监督学习的一种方法，利用算法进行自动归类。分类算法主要解决如何将一个数据分到几个确定类别中的一类里去。分类算法通常需要样本数据训练模型，再利用模型进行数据分类，所以是一种监督学习。

聚类利用算法进行自动归类，是无监督学习。通过聚类分析我们可以发现事物的内在规律。
